Al enviar una petición POST a http://localhost:3000/reportes con un JSON como {"tipo": "Infraestructura", "descripcion": "Daño en alumbrado público"}, el servidor lo agrega al array reportes y responde confirmando el registro. Luego, al consultar GET /reportes, el servidor devuelve una lista en formato JSON con todos los reportes almacenados hasta ese momento, cada uno con su id, tipo y descripción. Esto demuestra cómo un array en memoria puede funcionar como una base de datos temporal mientras el servidor está encendido.
