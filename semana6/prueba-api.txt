Al enviar una petición POST a http://localhost:3000/registro con el JSON {"nombre": "Maria", "mensaje": "Hola comunidad"}, el servidor responde con un objeto JSON que confirma la recepción de los datos: {"estado": "Datos recibidos", "nombre": "Maria", "mensaje": "Hola comunidad"}. Esto demuestra que express.json() permitió leer correctamente el cuerpo de la petición y que la ruta pudo acceder a los campos nombre y mensaje mediante req.body.
